## 摘要

智慧選卡器是一個網頁應用，幫助使用者比較商品在不同電商平台的價格，並推薦最適合的信用卡優惠。核心功能包括：商品網址解析、多平台價格爬取、信用卡優惠匹配、聯盟行銷連結生成。技術方案採用最簡捷且成本最低的方式：靜態網頁 + 免費網頁爬蟲服務 + JSON 資料儲存。

## 技術背景

**語言/版本**: HTML/CSS/JavaScript (純前端，無需伺服器)
**主要依賴**:
- 前端：原生 JavaScript (無框架)
- 爬蟲：免費第三方爬蟲 API（如 ScraperAPI 免費額度或 Cheerio.js）
- 資料儲存：瀏覽器 LocalStorage + 靜態 JSON 檔案

**儲存**: LocalStorage（瀏覽器本地儲存）+ 靜態 JSON 檔案（信用卡優惠資料）
**測試**: 手動測試（符合憲章的測試策略）
**目標平台**: 現代瀏覽器（Chrome, Firefox, Safari, Edge）
**專案類型**: 單一網頁應用（Single Page Application）
**效能目標**:
- 頁面載入 < 2 秒
- 商品解析 + 價格查詢 < 10 秒（符合規格 SC-001）
- 支援 100 名同時使用者（靜態網頁無伺服器負載問題）

**限制**:
- 免費爬蟲 API 限制（每日請求次數，需研究具體服務）
- 無後端資料庫（資料持久化僅限 LocalStorage）
- 信用卡優惠資料需手動維護更新

**規模/範圍**:
- MVP 階段支援 3-5 個主流電商平台
- 維護 10-20 張信用卡優惠資料
- 預期初期使用者 < 1000 人/月

### ✅ MVP 優先
- **通過**: 採用純前端方案，無需後端伺服器，快速開發和部署
- **通過**: 使用免費服務和工具，零成本啟動
- **通過**: 核心功能（比價 + 信用卡推薦）優先，進階功能（P3 最佳組合）可後續迭代

- **通過**: 靜態網頁部署快速（GitHub Pages 或 Netlify 免費託管）
- **通過**: 原生 JavaScript 無需打包工具，開發流程簡單
- **通過**: 資料維護（信用卡優惠）可透過更新 JSON 檔案快速調整
